
-- SUMMARY --

This module adds a new field to all content types, "Last Updated", and adds a
new panel pane to override the node add/edit form. Also it contains an update
script for modifying all node titles to contain their author information.

-- REQUIREMENTS --

Ctools, Page Manager, Panels, Features.

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/895232 for further information.
